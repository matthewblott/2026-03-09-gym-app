# Project Brief

I have a Rails application for a gym app. with the following features:
- It is multi tenant and the Apartment library is used for this purpose.
- Users can sign in as guest with their session tied to that device.
- When a guest signs out all their data from that sessions is deleted along with the guest account created.
- Users can add an email to their guest account which allows them to sign in on other devices.
- One time password functionality is used for the signing in process for email accounts.
